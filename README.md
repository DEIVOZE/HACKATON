Инструкция по запуску:
1. Установить python версии 3.10 и выше
2. Установиь необходимые библиотеки из requirements.txt
3. Для корректной работы необходимо установить ffmpeg и добавить его в виртуальное окружение
4. Скачать архив проекта и разархивировать его в любой папке (желательно без русских букв в пути
5. Запустить файл main.py
6. Перейти по ссылке http://127.0.0.1:8000/
